# Tyler Nguyen Portfolio

A responsive personal portfolio built with semantic HTML, modern CSS, and
vanilla JavaScript. The site uses Vite for local development and production
builds.

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run preview
```

The generated site is written to `dist/` and can be deployed directly to
Vercel, Netlify, or any static hosting provider.
